# 郭楠大冒险

傻逼群友想玩儿的。做个试试。

> 从 Outext 抄来的框架。<a href="https://github.com/mrleidesen/Outext" target="_blank" >https://github.com/mrleidesen/Outext</a>

## 规则

每次选择都会有时间限制（时间长短不一），如果超时未选，会直接死亡

## 运行

```shell
npm install -g pnpm

pnpm install

pnpm dev
```

## 开发

在 `src/data/index.ts` 里面有剧情信息。基本上只要改这里即可。  
提交代码，合并到 `main` 分支即可自动部署。
